 Cara Backup PostgreSQL Saat Ini (Docker Compose)
 
*Backup dari container
 
docker exec -t finote-postgres \
pg_dump -U finote_user -d finote \
> finote_backup.sql

terbentuk : finote_backup.sql

*backup all databse 

docker exec -t finote-postgres \
pg_dumpall -U finote_user \
> finote_full_backup.sql

*Copy dari container (optional method)

docker exec finote-postgres \
pg_dump -U finote_user finote \
> backup.sql


---

Restore ke Kubernetes

Jika sudah deploy PostgreSQL di K8s:
kubectl exec -i postgres-0 -- \
psql -U finote_user -d finote < finote_backup.sql

atau jika pakai service
kubectl port-forward svc/postgres 5432:5432
psql -h localhost -U finote_user -d finote < finote_backup.sql


Urutan aman:

backup DB dari compose
deploy postgres di k8s
restore backup
jalankan migration job (opsional)
deploy backend
test koneksi
update frontend ingress ke backend baru
